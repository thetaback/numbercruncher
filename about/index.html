About

Number Cruncher is an educational game designed for use in the classroom as a fun and engaging way to practice arithmetic skills.

Designed & Developed By Theta Back

Technologies Used

Number Cruncher uses the following:

  • ct.js, a game engine
  • Docker, a container engine
  • Bootswatch's Solar theme for Bootstrap
  • Bootstrap, a frontend toolkit for styling
  • Pocketbase, a backend API and database
  • Vanilla JS, for UI logic
  • HTML, for frontend presentation
